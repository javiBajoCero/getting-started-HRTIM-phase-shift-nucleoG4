# getting-started-HRTIM-phase-shift-nucleoG4
CubeIDE project containing the scheleton code to fire phase shifted signals for Dual Active Bridge control.

Code is based on page 34 from [an4539 (Multiphase buck converter)](https://www.st.com/resource/en/application_note/an4539-hrtim-cookbook-stmicroelectronics.pdf)    
Progress was documented here: [st community question](https://community.st.com/t5/stm32-mcus-products/synching-hrtim-s-tima-and-timb/td-p/770972)
